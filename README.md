# Multi-Tenant Clinic Management System

A secure, production-ready clinic management system built with Next.js, Firebase, and TypeScript.

## Features

✅ **Multi-Tenant Architecture** - Complete clinic isolation  
✅ **Patient Self-Booking** - Public booking with QR code status tracking  
✅ **Staff Management** - Doctor and nurse profiles with role-based access  
✅ **Queue Management** - Real-time queue updates and patient status tracking  
✅ **Security Hardened** - Server-side validation, anti-scraping, strict Firestore rules
