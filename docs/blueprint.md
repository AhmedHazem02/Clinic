# **App Name**: QueueWise Clinic

## Core Features:

- Patient Registration: Nurse Panel: Form for registering new patients (Name, Phone, Booking Date, Diseases, Age), automatically assigning queue numbers and generating QR codes.
- Patient Queue View: Patient Panel: Allows patients to access their queue information by scanning a QR code or entering their phone number.
- Doctor Dashboard: Doctor Panel: Interface for doctors to manage their availability, view current patient details, and access patient data.
- AI-Assisted Prescriptions: Generate suggested phrases for prescriptions with tool-assisted prescription recommendations. Doctors can customize these phrases as needed.
- Real-Time Queue Display: Displays all patients in queue with real-time updates, including waiting times (15 minutes per patient), dynamically updated as the queue changes.
- User Authentication: Nurses and Doctors login using email and password

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to convey trust and reliability in the medical setting.
- Background color: Light gray (#F0F4F8), nearly desaturated blue, for a clean, calming backdrop.
- Accent color: Muted purple (#807592), for highlighting interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headers and 'Inter' (sans-serif) for body text, for a blend of modern and readable design.
- Use consistent and clear icons from a Material Design or similar library for queue status, patient info, and panel navigation.
- A clean, card-based layout with clear information hierarchy is expected.
- Smooth transitions and subtle animations (e.g., queue updates, form submissions) to enhance user experience.
